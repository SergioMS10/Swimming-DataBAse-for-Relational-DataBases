# DBTermProject - Database Term Project Main Folder

This is a folder for developing your database term project using DataGrip (or Visual Studio Code)

Note that there are various sub-folders. You will be using these to deliver various intermediate pieces of your database term project.

| Folder or File                 | Description                                                                                                                                                         |
|--------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Data-Files                     | In this folder you will place your spreadsheets, CSV files, or other raw data files.                                                                                |
| Data-Model                     | When you have made a diagram of your data model, place the image file and the description document here                                                             |
| script_order.txt               | When you write scripts to create and load your database, you will list them in this file.                                                                           |
| SQL-DB-Creation-Scripts        | Here you will create SQL files that create your database. This will include the trigger(s) and/or stored procedure(s) you create with the final project deliverable |
| SQL-Business-Operation-Scripts | You will place your business queries, business updates,                                                                                                             |                                                                                                            |

Consult the posted materials for choosing and implementing your project, and enjoy it!
