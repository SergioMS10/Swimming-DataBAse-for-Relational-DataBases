# Data-Files

In this folder you will store your initial data that you will use for your project. This may be one or more Excel spreadsheets, or raw data in comma-separated (".csv") or other textual formats.

Consult the assignments for detailed instructions.

You will _not_ need to update these files after your initial delivery, because you will be putting your data into SQL scripts.

However if you find it useful to update these files as your project progresses, that is perfectly acceptable.
